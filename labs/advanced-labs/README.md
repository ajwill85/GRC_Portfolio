# Advanced Labs

This directory contains references to advanced GRC portfolio labs. These labs are designed to be more complex and comprehensive than the basic labs, and are intended for experienced practitioners looking to showcase advanced skills.

Due to their size and complexity, these advanced labs are hosted in separate repositories. Below you'll find links and descriptions for each advanced lab.

## Available Advanced Labs

### [AWS Automated Access Review](https://github.com/ajy0127/aws_automated_access_review)

AWS Access Review is a comprehensive, zero-configuration security assessment tool that automatically evaluates your AWS environment for potential security risks and compliance gaps.

This lab demonstrates how to implement automated access reviews in AWS environments to maintain security posture and compliance with regulatory frameworks.
